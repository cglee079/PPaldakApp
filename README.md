<h2><strong>개요</strong></h2>

<p>본 프로젝트는 &nbsp;IOT 스마트 낚싯대 프로젝트 입니다.<br/>
기존에 낚싯대에 아두이노를 부착하여 블루투스를 통하여 스마트폰과 통신합니다.<br/>
주요기능으로는 무게 센서를 통하여, 사용자에게 스마트폰을 통해 입질 알림을 주고<br/>
물고기에 힘을 측정하여 서버에 저장 하여 추후 확인, 공유, 랭킹화가 가능 합니다.</p>
